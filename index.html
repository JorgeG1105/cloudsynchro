CloudSynchro Hosting FAQs

What is CloudSynchro Hosting?

CloudSynchro Servers are different from traditional hosting. With a CloudSynchro Server, your website is not hosted on a single server, it is hosted across several different servers. Each server contains your website and the server(s) are connected via the internet.

How do I buy a CloudSynchro Server?

Discovering the right fit of a cloud server for your organizations requires time and research. Before you purchase, review the needs of your business obligations. For example, your business may have several security or government requirements. Several organizations have policies in place which do not allow monthly recurring charges for renting a machine. Once you have the proper breakdown of what you need out of your server, contact an InMotion Hosting sales rep via phone or chat and we can build a custom hosting plan specifically for your company.

What is the difference between Cloud VPS and Managed VPS?

Self-managed Cloud VPS has cheaper options but does not have cPanel nor other graphical user interface (GUI) software. Cloud Server Hosting plans range from $5-$160/monthly with multiple scaling options, so you only pay for the resources you need. You can choose between three operating systems (OSs) – CentOS, Debian, and Ubuntu – and re-OS your Cloud Server at any time. However, there’s no cPanel or other graphical user interface (GUI) for managing the server – just SSH. Server OS updates, backups (excluding AMP snapshots), and other server management tasks are your responsibility. You’re creating your web hosting solution from scratch on a bare server OS, and have complete control over applications installed. Learn more about the differences between Cloud VPS and Managed VPS.
